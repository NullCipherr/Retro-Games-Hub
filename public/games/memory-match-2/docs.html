Memory Match 2.0

Jogo de memória com foco em ritmo, eficiência de jogadas e melhor tempo.

Objetivo

Encontrar todos os pares do tabuleiro no menor tempo, com o menor número de jogadas possível.

Mecânicas

  • 8 pares de cartas (16 cartas no total)
  • Contador de jogadas e sequência de acertos
  • Melhor tempo salvo no navegador

Controles

  • Clique/toque para virar cartas
  • R para reiniciar a rodada
