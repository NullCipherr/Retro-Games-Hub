# Retro Games Hub 🎮

Uma coleção premium de jogos clássicos com visual neon futurista, organizada para produção e pronta para o GitHub Pages.

## 🏗 Estrutura do Projeto

- **index.html**: O Hub principal que centraliza todos os jogos.
- **assets/**: Contém todos os recursos globais (CSS, JS).
- **games/**: Contém as pastas individuais de cada jogo.

## 🚀 Como Jogar

1. Basta abrir o `index.html` em qualquer navegador moderno.
2. Ou acesse através do link do GitHub Pages (se configurado).
