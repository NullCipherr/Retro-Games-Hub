# 💣 Neon Minesweeper

Uma versão moderna e estilizada do clássico Campo Minado.

## 🎮 Características

### Gameplay
- **Geração Procedural**: Mapas gerados aleatoriamente a cada partida
- **Algoritmo de Flood Fill**: Abertura automática de áreas vazias
- **Sistema de Bandeiras**: Marque minas suspeitas com clique direito
- **3 Dificuldades**: Fácil (10x10), Médio (15x15) e Difícil (20x20)
- **Primeiro Clique Seguro**: Garante que você nunca exploda na primeira jogada

### Visual
- **Estética Neon**: Cores vibrantes para números e elementos
- **Interface Limpa**: Design minimalista com foco no tabuleiro
- **Animações**: Feedback visual suave

## 🕹️ Controles

| Ação | Comando |
|------|---------|
| Revelar Célula | `Clique Esquerdo` |
| Colocar Bandeira | `Clique Direito` |
| Reiniciar | `R` |
