# 🚀 Neon Space Invaders

Um jogo clássico de Space Invaders reimaginado com visual neon moderno e premium.

## 🎮 Características

### Gameplay
- **Mecânica Clássica**: Controle sua nave e destrua ondas de invasores alienígenas
- **Sistema de Ondas**: Dificuldade progressiva com cada onda derrotada
- **Barreiras Destrutíveis**: Use barreiras de proteção estrategicamente
- **Nave Misteriosa**: Destrua UFOs bônus para pontos extras
- **Sistema de Vidas**: 3 vidas iniciais + vida bônus a cada onda
- **High Score**: Sistema de pontuação máxima salvo localmente

### Visual
- **Estética Neon**: Design cyberpunk com efeitos de brilho neon
- **Efeitos de Partículas**: Explosões animadas ao destruir inimigos
- **Glassmorphism**: Painéis com efeito de vidro fosco
- **Animações Suaves**: Transições e movimentos fluidos
- **Background Animado**: Estrelas e silhuetas de aliens na tela inicial

### Recursos Técnicos
- **100% Front-end**: HTML5, CSS3 e JavaScript puro
- **Canvas API**: Renderização otimizada
- **Responsive**: Adaptável para diferentes tamanhos de tela
- **LocalStorage**: Salvamento de high score
- **Sem Dependências**: Não requer bibliotecas externas

## 🕹️ Controles

| Tecla | Ação |
|-------|------|
| `←` | Mover para Esquerda |
| `→` | Mover para Direita |
| `Space` | Atirar |
| `P` | Pausar/Despausar |

## 📊 Sistema de Pontuação

| Inimigo | Pontos |
|---------|--------|
| Invasor Tipo 1 (Inferior) | 10 pontos |
| Invasor Tipo 2 (Médio) | 20 pontos |
| Invasor Tipo 3 (Superior) | 30 pontos |
| Nave Misteriosa | 50-100 pontos |

## 🚀 Como Jogar

1. Abra `index.html` no navegador
2. Clique em "INICIAR MISSÃO"
3. Use as setas para mover sua nave
4. Pressione Space para atirar
5. Destrua todos os invasores antes que eles atinjam sua posição
6. Complete ondas para aumentar seu score

## 🎯 Estratégias

1. **Elimine as Laterais**: Destruir colunas laterais reduz a velocidade de movimento
2. **Use as Barreiras**: Proteja-se dos disparos inimigos, mas não fique parado
3. **Mire nos Superiores**: Invasores do topo valem mais pontos
4. **Fique em Movimento**: Evite ser atingido mantendo-se ágil
5. **Economize Munição**: Cada tiro conta!
