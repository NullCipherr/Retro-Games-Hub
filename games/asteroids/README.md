# ☄️ Neon Asteroids

Uma recriação moderna do clássico Asteroids com estética neon e física vetorial.

## 🎮 Características

### Gameplay
- **Física de Inércia**: Controle realista de nave no vácuo
- **Wrap-around**: Tela infinita onde bordas conectam aos lados opostos
- **Fragmentação**: Asteroides se dividem em pedaços menores
- **Dificuldade Progressiva**: Mais asteroides e maior velocidade a cada nível
- **High Score**: Salvo localmente

### Visual
- **Estética Neon**: Vetores brilhantes e rastro de luz
- **Partículas**: Explosões coloridas ao destruir asteroides
- **60 FPS**: Animação fluida e responsiva

## 🕹️ Controles

| Tecla | Ação |
|-------|------|
| `↑` | Acelerar (Propulsor) |
| `← / →` | Girar Nave |
| `Space` | Atirar |
| `P` | Pausar |
