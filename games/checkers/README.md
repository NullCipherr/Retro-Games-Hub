# ♟️ Neon Checkers

O clássico jogo de Damas com uma estética neon futurista.

## 🎮 Características

### Gameplay
- **Regras Oficiais**: Movimento diagonal, captura obrigatória e promoção a Dama.
- **Captura Múltipla**: O jogo permite (e obriga) sequências de capturas.
- **Multiplayer Local**: Jogue contra um amigo no mesmo dispositivo.
- **Turnos Claros**: Indicador visual de quem é a vez.

### Visual
- **Estética Neon**: Peças Cyan vs Pink com brilho intenso.
- **Damas Reais**: Peças promovidas ganham uma coroa dourada.
- **Feedback Visual**: Destaque para movimentos possíveis e peças selecionadas.

## 🕹️ Controles

| Ação | Comando |
|------|---------|
| Selecionar Peça | `Clique` |
| Mover Peça | `Clique` na casa destino |
| Cancelar Seleção | `Clique` em outra peça ou no vazio |
